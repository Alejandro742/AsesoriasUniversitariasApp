Bueno dejando eso atrás ahora sí con la parte de preparar todo para el funcionamiento del proyecto.

- Para que pueda funcionar el proyecto primero deberá tener "Node" (adjunto link: https://nodejs.org/en/).
- Tendrá que abrir una terminal en donde está guardado el proyecto y ejecutar el comando: "npm install", esto instalará 
  las librerías que utilizé en el proyecto.
- Una vez ya instalado todo solo hace falata ejecutar el siguiente comando: "npm run dev" para inicializar el servidor local
  y en terminal aparecerá que el servidor está escuchando en localhost 3000.

Hay un archivo que se llama keys.js donde viene la información de la base de datos a donde se conectará ahí deberá cambiar la información
que crea necesaria, como el usuario o contraseña.

NOTA: * NO CIERRE LA TERMINAL QUE ABRIÓ PARA INICIAR EL SERVIDOR CON LOS COMANDOS DE ARRIBA YA QUE ESTO MATARÁ LA CONEXIÓN *
